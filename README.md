## デザインパターンとは
プログラムの設計において、定石となる手法をパターン化したものです。有名なものにGoFの23パターンがある。

GoFのパターンは20年以上前のものだし、言語レベルで実装済みのものもあるけど、OOPを理解するには知っておいてもいい？

あくまで、有用そうなのをパターン化しただけです。

## デザインパターンの種別
- 生成に関するパターン
- 構造に関するパターン
- 振る舞いに関するパターン
